#时光之旅
工业化，严格保护的几大产粮基地。未来地球污染，人类减少室外活动。为减少人类活动资源消耗，虚拟游戏蓬勃发展，vr逐渐成熟，开始探索。
减少人口。教育，娱乐需求能够完全满足。
人们一直探索更多的生存可能。但只有在游戏里。才能获取想要的消费品。
主角觉得这样自暴自弃，娱乐至死的价值观和自己有偏差。开始探索外部世界。然而想要探索外部世界，
游戏称谓最受欢迎的排解方式。为了能更好地把控每个人的生活动向。


*未来世界* *游戏*  *公平竞技* *谋略* *团队*
